> **关键对比**：2026Q1假设占比与招股书历史数据存在显著差异——
> - **DRAM**：2026Q1假设33%（Q1业绩说明会数据），但2025H1仅15.4%。Q1占比大幅提升是因为DRAM在2026年爆发式放量，33%是Q1时点值而非全年水平
> - **NOR Flash**：2026Q1假设40%，低于2025H1的49%。这是DRAM占比提升后的必然结果——DRAM从15.4%升至33%，挤占了其他板块份额，NOR占比被动下降是合理的
> - **MCU**：2026Q1假设20%，低于2025H1的23.1%。同理，DRAM放量导致MCU占比被动稀释
> - **NAND**：2026Q1假设7%，高于2025H1的4.2%，可能高估
>
> **注意**：2026Q1假设是时点结构（DRAM已放量），招股书是2025H1历史结构（DRAM尚未放量），两者不可直接比较。但frog用Q1占比推全年存在偏差——全年DRAM占比将介于15.4%（2025H1）和33%（2026Q1）之间，取决于后续季度放量节奏。

**Q1各板块营收推算（独立估算+剩余分配法）**：

存储业务爆发式增长，传感器和模拟有独立趋势，不应被DRAM放量被动稀释。因此采用"先独立估算非存储板块，再分配剩余"的方法：

**第一步：独立估算传感器和模拟Q1营收**

用(2025全年 - 2025H1) / 2 估算2025H2季度均值，作为Q1基准：

| 板块 | 2025全年 | 2025H1 | 2025H2 | H2季度均值 | Q1趋势修正 | 2026Q1估算 | 来源 |
|------|---------|--------|--------|-----------|-----------|-----------|------|
| 传感器 | 3.89亿 | 1.93亿 | 1.96亿 | **0.98亿** | 持平或萎缩 | **~0.98亿**（2.3%） | **[年报-P17]**；**[招股书-P8/P190/P253]** |
| 模拟 | 3.33亿 | 1.52亿 | 1.81亿 | **0.91亿** | 自有环比+15% | **~1.05亿**（2.5%） | **[年报-P17]**；**[招股书-P8/P190/P253]**；**[Q1业绩说明会-20260520]** |

> 模拟Q1修正：自有模拟收入环比+15% **[Q1业绩说明会-20260520]**，苏州赛芯毛利率50%+。0.91 × 1.15 ≈ 1.05亿

**第二步：计算存储+MCU剩余营收**

```
Q1总收入 = 41.88亿 [Q1报-P1/P8]
传感器 + 模拟 = 0.98 + 1.05 = 2.03亿
剩余（存储 + MCU）= 41.88 - 2.03 = 39.85亿
```

**第三步：在剩余中分配DRAM、NOR、NAND、MCU**

```
DRAM = 41.88 × 1/3 = 13.96亿 [业绩说明会-20260520]
NOR + NAND + MCU = 39.85 - 13.96 = 25.89亿
```

用2025H1内部比例分配NOR+NAND+MCU（25.89亿）：

| 板块 | 2025H1内部比例 | Q1推算营收 | 占总收入比 | frog假设占比 | 偏差 |
|------|--------------|-----------|----------|------------|------|
| NOR Flash | 64.2%（=49.0/76.3） | **16.62亿** | **39.7%** | 40% | +0.3pp |
| MCU | 30.3%（=23.1/76.3） | **7.84亿** | **18.7%** | 20% | +1.3pp |
| SLC NAND | 5.5%（=4.2/76.3） | **1.43亿** | **3.4%** | 7% | +3.6pp |

**完整Q1结构推算 vs frog假设**：

| 板块 | Q1推算营收 | 推算占比 | frog假设占比 | 偏差 | 说明 |
|------|-----------|---------|------------|------|------|
| DRAM | 13.96亿 | 33.3% | 33% | — | 业绩说明会锚定 |
| NOR Flash | 16.62亿 | 39.7% | 40% | +0.3pp | 接近 |
| MCU | 7.84亿 | 18.7% | 20% | +1.3pp | frog略高估 |
| SLC NAND | 1.43亿 | 3.4% | 7% | +3.6pp | frog显著高估 |
| 传感器 | 0.98亿 | 2.3% | — | 遗漏 | 萎缩板块 |
| 模拟 | 1.05亿 | 2.5% | — | 遗漏 | 增长板块 |
| **合计** | **41.88亿** | **100%** | **100%** | | |

> **核心结论**：用独立估算法替代等比例稀释法后，NOR推算占比39.7%与frog假设40%几乎一致，MCU推算18.7%与frog假设20%偏差缩小。关键差异仍在NAND：frog假设7%但推算仅3.4%，高估3.6pp。frog遗漏的传感器（2.3%）和模拟（2.5%）合计4.8%，这部分"消失"的占比被frog分配到了NAND和其他板块中。

| 板块 | 2025年营收 | 趋势 | 2026年判断 | 来源 |
|------|-----------|------|-----------|------|

### 2.4 非 DRAM 一季度毛利率 48.22% 的来源

**48.22% 在两份财报中均未直接披露，是推算值。**

推算依据来自2025年报"主营业务分产品情况"表：

| 产品 | 收入（亿） | 成本（亿） | 毛利率 | 来源 |
|------|-----------|-----------|--------|------|
| 存储芯片（含DRAM+NOR+SLC NAND） | 65.66 | 37.53 | **42.84%** | **[年报-P17]** 主营业务分产品情况表 |
| 微控制器（MCU） | 19.10 | 12.26 | 35.82% | **[年报-P17]** 同上 |
| 传感器 | 3.89 | 3.13 | 19.57% | **[年报-P17]** 同上 |
| 模拟产品 | 3.33 | 2.10 | 36.96% | **[年报-P17]** 同上 |
| 整体 | 92.02 | 55.02 | 40.21% | **[年报-P17]** 主营业务分行业情况表 |

推算方程：

```
Flash毛利率 = (存储芯片毛利 - DRAM毛利) / (存储芯片收入 - DRAM收入)
```

代入参数：

| 参数 | 数值 | 依据 |
|------|------|------|
| DRAM收入 | ~15.20亿 | 详见下方推算 |
| DRAM毛利率 | ~25% | **[年报-P11]** "自2025年第二季度起，公司利基型DRAM产品量价齐升，毛利率季度环比改善明显"，25%为推算的全年加权值（公开信息从未披露DRAM具体毛利率数值，仅有定性描述） |
| DRAM成本 | ~11.40亿 | **[推算]** 与长鑫关联采购11.82亿 **[年报-P19/P202]** 基本吻合 |

**2025年DRAM收入推算（~15.20亿）：**

2025全年DRAM收入在所有官方文件中均未单独披露，需通过多源数据推算。

**已知数据：**

| 来源 | 期间 | 存储芯片收入 | DRAM收入 | 来源标记 |
|------|------|------------|---------|---------|
| 招股书 | 2025H1 | 28.45亿 | **6.37亿** | **[招股书-P8/P190/P253]** |
| 招股书附录 | 2025年1-9月 | 47.68亿 | 未拆分 | **[招股书-附录一A]** |
| 年报 | 2025全年 | 65.66亿 | 未拆分 | **[年报-P17]** |
| Q3季报 | 2025年1-9月 | 47.68亿 | 未拆分 | **[Q3季报-P8]** |

**推算路径一：用50%增长目标**

```
2024年DRAM收入 = 10.73亿 [招股书-P8/P190/P253]
2025年DRAM营收同比增长50%目标 [Q3业绩说明会-20251028]
→ 2025年DRAM收入 ≈ 10.73 × 150% = 16.10亿
```

frog取~15.20亿，比16.10亿更保守。

**推算路径二：用长鑫关联采购反推**

```
2025年长鑫关联采购 = 11.82亿 [年报-P19/P202]
≈ DRAM晶圆采购成本

DRAM收入 = 采购成本 / (1 - DRAM毛利率)

DRAM毛利率推算（基于Q3业绩说明会定性描述 [Q3业绩说明会-20251028]）：
  Q1: "低个位数"   ≈ 5%
  Q2: "双位数"     ≈ 15%
  Q3: "较好水平"   ≈ 30%
  Q4: "健康毛利率" ≈ 40%
  全年加权 ≈ 25%（粗略算术平均）

毛利率25% → DRAM收入 = 11.82 / 0.75 = 15.76亿
毛利率30% → DRAM收入 = 11.82 / 0.70 = 16.89亿
毛利率35% → DRAM收入 = 11.82 / 0.65 = 18.18亿
```

**推算路径三：用存储芯片季度拆分验证**

```
Q3存储芯片 = 47.68 - 28.45 = 19.23亿
Q4存储芯片 = 65.66 - 47.68 = 17.98亿

2025H1 DRAM占存储芯片 = 6.37 / 28.45 = 22.4%
若全年DRAM占比维持22.4% → DRAM收入 = 65.66 × 22.4% = 14.71亿
若下半年DRAM占比提升至30% → DRAM收入 ≈ 6.37 + (19.23+17.98)×30% = 17.53亿
```

**三条路径交叉验证：**

| 推算路径 | 2025年DRAM收入 | 可信度 |
|----------|---------------|--------|
| 50%增长目标 | 16.10亿 | 高（公司官方目标） |
| 长鑫关联采购（毛利率25%） | 15.76亿 | 中（依赖毛利率假设） |
| 长鑫关联采购（毛利率30%） | 16.89亿 | 中 |
| 存储芯片占比（维持H1比例） | 14.71亿 | 低（低估下半年放量） |
| 存储芯片占比（下半年30%） | 17.53亿 | 中 |

**综合判断**：2025年DRAM收入约**15.2亿-16.9亿**，frog取~15.20亿偏保守但合理。

```
Flash毛利 = 65.66 - 37.53 - (15.20 - 11.40) = 24.33亿
Flash收入 = 65.66 - 15.20 = 50.46亿
Flash毛利率 = 24.33 / 50.46 ≈ 48.22%
```

> **校验提示**：48.22%的推算依赖两个数据：(1) H股招股书DRAM收入10.73亿（2024年）**[招股书-P8/P190/P253]**，1.464亿美元见 **[招股书-P122/P123]**；(2) 2025年DRAM营收同比增长50%目标 **[Q3业绩说明会-20251028]**。2025年DRAM毛利率无公开精确数值，仅有"低个位数→双位数→较好→健康"的定性轨迹 **[Q3业绩说明会-20251028]**。frog在 **[讨论-L148]** 和 **[讨论-L156]** 中使用了0.4822，在風里的我在 **[讨论-L172]** 中以"48.2%"提及。

**关键问题**：48.22%是2025全年Flash毛利率，frog直接用作Q1非DRAM毛利率基准，隐含假设Q1 Flash毛利与2025全年持平。实际上Q1 NOR已涨价，Flash毛利率应已高于2025全年水平。

## 三、公式推演流程（4步）

### 第1步：计算非DRAM板块的毛利额

```
非DRAM毛利 = 总收入 × 非DRAM占比 × 非DRAM毛利率
           = 41.88 × (2/3) × 0.4822
           = 13.463亿
```

**逻辑**：从总收入中切出非DRAM部分（2/3），再乘以非DRAM的毛利率，得到NOR + MCU + SLC NAND等板块贡献的毛利额。

**数据来源**：
- 41.88亿：**[Q1报-P1/P8]**
- 非DRAM占比2/3：由DRAM占比1/3 **[讨论-L230]** 推导
- 0.4822：**[讨论-L148/L156]** frog使用，推算来源见2.4节

### 第2步：反推DRAM板块的毛利额

```
DRAM毛利 = 总收入 × 综合毛利率 - 非DRAM毛利
         = 41.88 × 0.5708 - 13.463
         = 23.905 - 13.463
         = 10.442亿
```

**逻辑**：用总收入 × 综合毛利率得到总毛利额，再减去已算出的非DRAM毛利，剩余就是DRAM贡献的毛利。这是经典的**扣除法反推**。

**数据来源**：
- 0.5708：**[推算]** 由 **[Q1报-P8]** 营业收入和营业成本计算得出

### 第3步：反推DRAM的毛利率

```
DRAM毛利率 = DRAM毛利 ÷ DRAM收入
           = 10.442 ÷ (41.88 × 1/3)
           = 10.442 ÷ 13.96
           = 74.8%
```

**逻辑**：用反推出的DRAM毛利额除以DRAM收入（总收入 × 1/3），得到DRAM的毛利率。

**frog的调整**：认为74.8%偏高，因长鑫涨价会传导到下游，**保守下调到70%**。**[讨论-L200]** frog："长鑫会涨价"

### 第4步：用调整后的DRAM毛利率，预估全年DRAM毛利贡献

```
57.11 ÷ 5.81 × 10.44 ÷ 75 × 70 = 95.78
```

**原始公式位置**：**[讨论-L206]** frog直接给出 `57.11÷5.81×10.44÷75×70=95.77966265`

逐步拆解：

| 步骤 | 计算 | 含义 |
|------|------|------|
| ① | `57.11 ÷ 5.81 = 9.83` | 全年DRAM关联交易预计 **[关联交易公告-20260331]** ÷ Q1实际交付 **[关联交易公告-20260331]** = **年化放大倍数** |
| ② | `9.83 × 10.44 = 102.62` | 放大倍数 × Q1 DRAM毛利额 **[推算]** = **全年DRAM毛利（按75%毛利率）** |
| ③ | `102.62 ÷ 75 × 70 = 95.78` | 从75%毛利率调整到70% **[讨论]** = **全年DRAM毛利（保守估计）** |

**57.11 和 5.81 的含义**：
- **57.11亿**：2026年长鑫DRAM关联交易预计总额（全年预计采购额度）**[关联交易公告-20260331]**
- **5.81亿**：Q1实际交付的DRAM金额 **[关联交易公告-20260331]**
- `57.11 ÷ 5.81 ≈ 9.83`：全年DRAM采购规模是Q1交付的9.83倍

**为什么放大倍数是9.83而不是4**：如果简单年化（Q1 × 4），倍数应为4。实际9.83说明Q1交付量远低于全年平均水平，DRAM交付节奏**前低后高**，后续季度将大幅放量。

## 六、CPA精算视角：53.22% → 51.89% 的修正依据

frog从乐观版53.22% **[讨论-L256]** 调整为保守版51.89% **[讨论-L276]**，差值1.33pp（即从48.22%基础上加5pp改为加3.67pp）。从CPA/精算师角度，有以下会计规则和公式指导这种修正。

### 6.1 修正因素一：加权平均（板块结构修正）

**会计原则**：不同产品毛利率不同时，综合毛利率应按收入占比加权计算，不能简单取算术平均。

48.22%是Flash（NOR + SLC NAND）的毛利率，**不包含MCU**。frog将其当作整个非DRAM部分的毛利率，CPA会做加权修正：

| 板块 | 占非DRAM比重 | 基础毛利率 | 涨价后毛利率 | 来源 |
|------|-------------|-----------|------------|------|
| NOR + SLC NAND | 70.1%（47/67） | 48.22% | 48.22% + 5pp = 53.22% | 48.22%来源见2.4节 |
| MCU | 29.9%（20/67） | 35.82% | 35.82% + 5pp = 40.82% | **[年报-P17]** + **[讨论-L464~L466]** frog："mcu的毛利只有35上下" |

```
加权非DRAM毛利率 = 70.1% × 53.22% + 29.9% × 40.82%
                 = 37.31% + 12.21%
                 = 49.52%
```

MCU的低毛利（35.82% **[年报-P17]**）拉低了综合值，但49.52%仍低于51.89%，说明加权平均不是唯一修正因素。

### 6.2 修正因素二：时间加权实现（涨价生效期间折算）

**会计原则**：收入和成本应按权责发生制在对应期间确认。涨价非全年均匀生效，应按实际生效月份加权。

| 板块 | 涨价时点 | 全年有效月份 | 占全年比重 | 来源 |
|------|---------|------------|-----------|------|
| NOR | 3月起（部分型号） | 约10个月 | 10/12 = 83.3% | **[讨论-L250]** "nor涨了15个点" |
| MCU | Q2起 | 约9个月 | 9/12 = 75% | **[讨论-L250]** "mcu二季度也涨了15个点" |

如果5pp是全年生效的毛利改善，时间加权后：

```
NOR改善 = 5pp × 83.3% = 4.17pp
MCU改善 = 5pp × 75% = 3.75pp

加权改善 = 70.1% × 4.17pp + 29.9% × 3.75pp
         = 2.92pp + 1.12pp
         = 4.04pp

调整后非DRAM毛利率 = 48.22% + 4.04% = 52.26%
```

接近51.89%，但还差0.37pp。

### 6.3 修正因素三：分母效应（收入基数扩大稀释毛利百分比）

**会计原则**：毛利率 = 毛利额 / 收入额。当收入因涨价而增加时，分母变大，即使毛利额增加，百分比提升也会被稀释。

这是从53.22%调整到51.89%**最关键的修正因素**。

```
原始状态：
  收入 = P，成本 = C，毛利率 = (P-C)/P = 48.22%
  即 C/P = 51.78%

涨价后：
  - 全产业链涨价15%，兆易售价涨15%，新收入 = 1.15P
  - 上游代工成本涨约7%（代工厂分得2/3涨幅），新成本 = 1.07C = 1.07 × 0.5178P = 0.5540P
  - 新毛利 = 1.15P - 0.5540P = 0.5960P
  - 新毛利率 = 0.5960 / 1.15 = 51.83%
```

**51.83%与51.89%几乎完全吻合！**

**直觉解释**：简单加5pp（48.22% + 5% = 53.22%）是错误的，因为5pp是毛利额的增量占原始收入的比例，但涨价后收入基数已经从P变成了1.15P。正确的做法是重新计算 `(新收入 - 新成本) / 新收入`，分母变大导致毛利率百分比被稀释。

### 6.4 修正因素四：精算审慎原则（保守平滑）

**精算原则**：在不确定条件下，精算师应取保守估计值，预留安全边际。

frog原话："平滑一季度的毛利，算了相对比例" **[讨论-L498~L508]** ——在分母效应修正的基础上，再额外预留0.06pp的安全边际（51.83% → 51.89%的微小差异可能来自四舍五入或frog手动微调）。

### 6.5 综合修正框架

将三个修正因素叠加：

| 修正因素 | 会计/精算依据 | 对53.22%的影响 | 修正量 |
|----------|-------------|---------------|--------|
| **加权平均** | 不同产品毛利率不同，应加权计算 | MCU低毛利 **[年报-P17]** 拉低综合 | -3.7pp |
| **时间加权** | 涨价非全年生效，按实际月份折算 | 涨价滞后效应 **[讨论-L250]** | -0.8~-1.2pp |
| **分母效应** | 毛利率=毛利/收入，收入增大稀释百分比 | 涨价后收入基数变大 | -1.0~-1.5pp |
| **保守平滑** | 精算审慎原则，不确定时取保守值 | 安全边际 **[讨论-L498~L508]** | -0.3~0.5pp |

**核心结论**：frog从53.22% **[讨论-L256]** 调到51.89% **[讨论-L276]**，最关键的修正是**分母效应**——涨价后收入基数变大（+15%），但毛利额增加不到15%，导致毛利率百分比被稀释。用"售价涨15%、成本涨7%"的假设，计算结果恰好约51.83%，与51.89%几乎完全吻合。

frog说的"平滑一季度的毛利，算了相对比例" **[讨论-L498~L508]**，本质上就是在做这个分母修正——不是简单加5个百分点，而是重新计算 `(新收入 - 新成本) / 新收入`。

### 6.6 CPA精算修正公式总结

```
简单加法（错误）：
  新毛利率 = 原毛利率 + 涨价分润 = 48.22% + 5% = 53.22%  ❌

分母效应修正（正确）：
  新毛利率 = (新收入 - 新成本) / 新收入
           = (1.15P - 1.07C) / (1.15P)
           = (1.15 - 1.07×0.5178) / 1.15
           = (1.15 - 0.5540) / 1.15
           = 0.5960 / 1.15
           = 51.83%  ✅ ≈ 51.89%
```

## 七、推演方法核心特点

### 7.1 扣除法反推

先算已知的（非DRAM），再用总量减去已知部分得到未知的（DRAM）。这是信息不透明时最常用的拆分方法。

### 7.2 产业链涨价分润逻辑

涨价不是全部归兆易，而是按 **1/3归兆易、2/3归上游代工厂** 分配。假设基于"产业链共同增益"的认知。**[讨论]** frog假设，无财报数据支撑。

### 7.3 关联交易锚定年化

用长鑫关联交易（57.11亿全年预计 **[关联交易公告-20260331]** vs 5.81亿Q1实际 **[关联交易公告-20260331]**）计算年化放大倍数9.83，而非简单Q1×4。这揭示了DRAM业务**前低后高、加速放量**的节奏。

### 7.4 保守调整

- 机构数据给DRAM 74.8% → frog下调到70%
- 非DRAM毛利率从53.22% **[讨论-L256]** 回调到51.89% **[讨论-L276]**
- 体现"宁可低估不可高估"的思路

### 7.5 量增参数的废弃

原计划用8%量增预估，但因一季报有了新基数，直接用一季报数据打底，不再需要预估量增——从"预测"切换到"实证"。**[讨论-L378]** 在風里的我："然后量增8%"；**[讨论-L382]** frog："量增8这里我没算"

### 7.6 公式瑕疵的自我认知

frog明确说了"公式有瑕疵" **[讨论-L264]**——因为量能差异、涨价滞后效应、新业务未计入等因素，计算只是粗略估算。
