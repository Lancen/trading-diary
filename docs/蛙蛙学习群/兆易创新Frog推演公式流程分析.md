# 兆易创新 Frog 推演公式流程分析

> 基于蛙蛙学习群讨论记录，完整还原快乐的frog从一季报公开数据反推各业务板块毛利率的公式推导流程，并标注数据来源与假设条件。

---

## 一、推演目标

在缺乏细分报表（公司不单独披露DRAM/NOR/NAND各自收入和毛利率）的情况下，通过一季报公开数据 + 关联交易公告，**反推各业务板块毛利率，并预估全年业绩**。

### 2.4 非 DRAM 一季度毛利率 48.22% 的来源

**48.22% 在两份财报中均未直接披露，是推算值。**

推算依据来自2025年报"主营业务分产品情况"表：

| 产品 | 收入（亿） | 成本（亿） | 毛利率 |
|------|-----------|-----------|--------|
| 存储芯片（含DRAM+NOR+SLC NAND） | 65.66 | 37.53 | **42.84%** |
| 微控制器（MCU） | 19.10 | 12.26 | 35.82% |
| 传感器 | 3.89 | 3.13 | 19.57% |
| 模拟产品 | 3.33 | 2.10 | 36.96% |
| 整体 | 92.02 | 55.02 | 40.21% |

推算方程：

```
Flash毛利率 = (存储芯片毛利 - DRAM毛利) / (存储芯片收入 - DRAM收入)
```

代入参数：

| 参数 | 数值 | 依据 |
|------|------|------|
| DRAM收入 | ~15.20亿 | H股招股书2024年DRAM约10.5亿 + 业绩说明会50%增长目标 |
| DRAM毛利率 | ~25% | Q1低个位数→Q2双位数→Q3较好→Q4健康，全年加权约25% |
| DRAM成本 | ~11.40亿 | 与长鑫关联采购11.82亿基本吻合 |

```
Flash毛利 = 65.66 - 37.53 - (15.20 - 11.40) = 24.33亿
Flash收入 = 65.66 - 15.20 = 50.46亿
Flash毛利率 = 24.33 / 50.46 ≈ 48.22%
```

**关键问题**：48.22%是2025全年Flash毛利率，frog直接用作Q1非DRAM毛利率基准，隐含假设Q1 Flash毛利与2025全年持平。实际上Q1 NOR已涨价，Flash毛利率应已高于2025全年水平。

## 三、公式推演流程（4步）

### 第1步：计算非DRAM板块的毛利额

```
非DRAM毛利 = 总收入 × 非DRAM占比 × 非DRAM毛利率
           = 41.88 × (2/3) × 0.4822
           = 13.463亿
```

**逻辑**：从总收入中切出非DRAM部分（2/3），再乘以非DRAM的毛利率，得到NOR + MCU + SLC NAND等板块贡献的毛利额。

### 第2步：反推DRAM板块的毛利额

```
DRAM毛利 = 总收入 × 综合毛利率 - 非DRAM毛利
         = 41.88 × 0.5708 - 13.463
         = 23.905 - 13.463
         = 10.442亿
```

**逻辑**：用总收入 × 综合毛利率得到总毛利额，再减去已算出的非DRAM毛利，剩余就是DRAM贡献的毛利。这是经典的**扣除法反推**。

### 第3步：反推DRAM的毛利率

```
DRAM毛利率 = DRAM毛利 ÷ DRAM收入
           = 10.442 ÷ (41.88 × 1/3)
           = 10.442 ÷ 13.96
           = 74.8%
```

**逻辑**：用反推出的DRAM毛利额除以DRAM收入（总收入 × 1/3），得到DRAM的毛利率。

**frog的调整**：认为74.8%偏高，因长鑫涨价会传导到下游，**保守下调到70%**。

### 第4步：用调整后的DRAM毛利率，预估全年DRAM毛利贡献

```
57.11 ÷ 5.81 × 10.44 ÷ 75 × 70 = 95.78
```

逐步拆解：

| 步骤 | 计算 | 含义 |
|------|------|------|
| ① | `57.11 ÷ 5.81 = 9.83` | 全年DRAM关联交易预计 ÷ Q1实际交付 = **年化放大倍数** |
| ② | `9.83 × 10.44 = 102.62` | 放大倍数 × Q1 DRAM毛利额 = **全年DRAM毛利（按75%毛利率）** |
| ③ | `102.62 ÷ 75 × 70 = 95.78` | 从75%毛利率调整到70% = **全年DRAM毛利（保守估计）** |

**57.11 和 5.81 的含义**：
- **57.11亿**：2026年长鑫DRAM关联交易预计总额（全年预计采购额度）
- **5.81亿**：Q1实际交付的DRAM金额
- `57.11 ÷ 5.81 ≈ 9.83`：全年DRAM采购规模是Q1交付的9.83倍

**为什么放大倍数是9.83而不是4**：如果简单年化（Q1 × 4），倍数应为4。实际9.83说明Q1交付量远低于全年平均水平，DRAM交付节奏**前低后高**，后续季度将大幅放量。

## 四、涨价后的修正公式

### 4.1 修正逻辑

获取NOR和MCU涨价信息后，frog对非DRAM毛利率进行修正：

| 涨价信息 | 幅度 | 来源 |
|----------|------|------|
| NOR Flash涨价 | 15个点 | 一季度+二季度，二季度更凶 |
| MCU涨价 | 15个点（主流系列20%） | 二季度 |

**涨价分配假设（产业链共同增益）**：
- 上游代工厂（中芯国际）：分得 **2/3** 涨幅（约10个点）
- 下游兆易：分得 **1/3** 涨幅（约5个点）

### 4.2 修正公式（乐观版）

```
全年非DRAM毛利 = 总收入 × 非DRAM占比 × 修正后毛利率 × 年化倍数
               = 41.88 × (2/3) × 0.5322 × (12/3)
               = 59.44亿
```

| 参数 | 数值 | 说明 |
|------|------|------|
| 0.5322 | 48.22% + 5pp | 一季度基准毛利 + 1/3涨价空间 |
| 12/3 = 4 | 年化倍数 | 一季度 × 4 |

### 4.3 修正公式（保守版）

```
= 41.88 × (2/3) × 0.5189 × (12/3)
= 57.95亿
```

| 参数 | 数值 | 说明 |
|------|------|------|
| 0.5189 | 48.22% + 3.67pp | 更保守的涨价传导估计 |

### 4.4 在風里的我 对0.5189的理解

> "0.5189就是涨价后的单季营收减掉原来的营收，即新增毛利，然后用新毛利除Q1非DRAM营收得出的吧？"

frog的回应：

> "我用的是后续涨价逻辑，在一季度的预估毛利上加了1/3毛利空间，当做平均5个点，但是要平滑一季度的毛利，所以算了相对比例，这是不准确的，因为量能不一样，但是粗略算的"

## 六、CPA精算视角：53.22% → 51.89% 的修正依据

frog从乐观版53.22%调整为保守版51.89%，差值1.33pp（即从48.22%基础上加5pp改为加3.67pp）。从CPA/精算师角度，有以下会计规则和公式指导这种修正。

### 6.1 修正因素一：加权平均（板块结构修正）

**会计原则**：不同产品毛利率不同时，综合毛利率应按收入占比加权计算，不能简单取算术平均。

48.22%是Flash（NOR + SLC NAND）的毛利率，**不包含MCU**。frog将其当作整个非DRAM部分的毛利率，CPA会做加权修正：

| 板块 | 占非DRAM比重 | 基础毛利率 | 涨价后毛利率 |
|------|-------------|-----------|------------|
| NOR + SLC NAND | 70.1%（47/67） | 48.22% | 48.22% + 5pp = 53.22% |
| MCU | 29.9%（20/67） | 35.82% | 35.82% + 5pp = 40.82% |

```
加权非DRAM毛利率 = 70.1% × 53.22% + 29.9% × 40.82%
                 = 37.31% + 12.21%
                 = 49.52%
```

MCU的低毛利（35.82%）拉低了综合值，但49.52%仍低于51.89%，说明加权平均不是唯一修正因素。

### 6.2 修正因素二：时间加权实现（涨价生效期间折算）

**会计原则**：收入和成本应按权责发生制在对应期间确认。涨价非全年均匀生效，应按实际生效月份加权。

| 板块 | 涨价时点 | 全年有效月份 | 占全年比重 |
|------|---------|------------|-----------|
| NOR | 3月起（部分型号） | 约10个月 | 10/12 = 83.3% |
| MCU | Q2起 | 约9个月 | 9/12 = 75% |

如果5pp是全年生效的毛利改善，时间加权后：

```
NOR改善 = 5pp × 83.3% = 4.17pp
MCU改善 = 5pp × 75% = 3.75pp

加权改善 = 70.1% × 4.17pp + 29.9% × 3.75pp
         = 2.92pp + 1.12pp
         = 4.04pp

调整后非DRAM毛利率 = 48.22% + 4.04% = 52.26%
```

接近51.89%，但还差0.37pp。

### 6.3 修正因素三：分母效应（收入基数扩大稀释毛利百分比）

**会计原则**：毛利率 = 毛利额 / 收入额。当收入因涨价而增加时，分母变大，即使毛利额增加，百分比提升也会被稀释。

这是从53.22%调整到51.89%**最关键的修正因素**。

```
原始状态：
  收入 = P，成本 = C，毛利率 = (P-C)/P = 48.22%
  即 C/P = 51.78%

涨价后：
  - 全产业链涨价15%，兆易售价涨15%，新收入 = 1.15P
  - 上游代工成本涨约7%（代工厂分得2/3涨幅），新成本 = 1.07C = 1.07 × 0.5178P = 0.5540P
  - 新毛利 = 1.15P - 0.5540P = 0.5960P
  - 新毛利率 = 0.5960 / 1.15 = 51.83%
```

**51.83%与51.89%几乎完全吻合！**

**直觉解释**：简单加5pp（48.22% + 5% = 53.22%）是错误的，因为5pp是毛利额的增量占原始收入的比例，但涨价后收入基数已经从P变成了1.15P。正确的做法是重新计算 `(新收入 - 新成本) / 新收入`，分母变大导致毛利率百分比被稀释。

### 6.4 修正因素四：精算审慎原则（保守平滑）

**精算原则**：在不确定条件下，精算师应取保守估计值，预留安全边际。

frog原话："平滑一季度的毛利，算了相对比例"——在分母效应修正的基础上，再额外预留0.06pp的安全边际（51.83% → 51.89%的微小差异可能来自四舍五入或frog手动微调）。

### 6.5 综合修正框架

将三个修正因素叠加：

| 修正因素 | 会计/精算依据 | 对53.22%的影响 | 修正量 |
|----------|-------------|---------------|--------|
| **加权平均** | 不同产品毛利率不同，应加权计算 | MCU低毛利拉低综合 | -3.7pp |
| **时间加权** | 涨价非全年生效，按实际月份折算 | 涨价滞后效应 | -0.8~-1.2pp |
| **分母效应** | 毛利率=毛利/收入，收入增大稀释百分比 | 涨价后收入基数变大 | -1.0~-1.5pp |
| **保守平滑** | 精算审慎原则，不确定时取保守值 | 安全边际 | -0.3~0.5pp |

**核心结论**：frog从53.22%调到51.89%，最关键的修正是**分母效应**——涨价后收入基数变大（+15%），但毛利额增加不到15%，导致毛利率百分比被稀释。用"售价涨15%、成本涨7%"的假设，计算结果恰好约51.83%，与51.89%几乎完全吻合。

frog说的"平滑一季度的毛利，算了相对比例"，本质上就是在做这个分母修正——不是简单加5个百分点，而是重新计算 `(新收入 - 新成本) / 新收入`。

### 6.6 CPA精算修正公式总结

```
简单加法（错误）：
  新毛利率 = 原毛利率 + 涨价分润 = 48.22% + 5% = 53.22%  ❌

分母效应修正（正确）：
  新毛利率 = (新收入 - 新成本) / 新收入
           = (1.15P - 1.07C) / (1.15P)
           = (1.15 - 1.07×0.5178) / 1.15
           = (1.15 - 0.5540) / 1.15
           = 0.5960 / 1.15
           = 51.83%  ✅ ≈ 51.89%
```

## 七、推演方法核心特点

### 7.1 扣除法反推

先算已知的（非DRAM），再用总量减去已知部分得到未知的（DRAM）。这是信息不透明时最常用的拆分方法。

### 7.2 产业链涨价分润逻辑

涨价不是全部归兆易，而是按 **1/3归兆易、2/3归上游代工厂** 分配。假设基于"产业链共同增益"的认知。

### 7.3 关联交易锚定年化

用长鑫关联交易（57.11亿全年预计 vs 5.81亿Q1实际）计算年化放大倍数9.83，而非简单Q1×4。这揭示了DRAM业务**前低后高、加速放量**的节奏。

### 7.4 保守调整

- 机构数据给DRAM 74.8% → frog下调到70%
- 非DRAM毛利率从53.22%回调到51.89%
- 体现"宁可低估不可高估"的思路

### 7.5 量增参数的废弃

原计划用8%量增预估，但因一季报有了新基数，直接用一季报数据打底，不再需要预估量增——从"预测"切换到"实证"。

### 7.6 公式瑕疵的自我认知

frog明确说了"公式有瑕疵"——因为量能差异、涨价滞后效应、新业务未计入等因素，计算只是粗略估算。

## 八、推演局限与修正方向

| 局限 | 说明 | 修正方向 |
|------|------|---------|
| 48.22%基准偏差 | 用2025全年Flash毛利率当作Q1非DRAM基准，Q1 NOR已涨价 | 需Q1分板块数据修正 |
| DRAM占比用Q1推全年 | Q1 DRAM占比33%，但全年关联交易揭示占比将升至55%-60% | 用长鑫57.11亿关联交易修正全年占比 |
| 涨价1/3分润是假设 | 实际代工涨价比例未知，frog自认"公式有瑕疵" | 需代工成本数据验证 |
| 量能差异未处理 | 各板块量能不一样，相对比例平滑存在误差 | 需细分出货量数据 |
| 新业务未计入 | 车规放量、新样品取代eMMC未计入 | 需半年报补充 |
| 一季度毛利未合上 | MCU毛利35%与整体毛利存在矛盾 | 需细分报表验证 |
