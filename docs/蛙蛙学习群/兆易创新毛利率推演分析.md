## 二、聊天中的推演逻辑还原

### 2.1 frog的核心推演思路

frog的推演分为三个层次：

#### 第一层：整体毛利率预判

frog看到一季报数据后，按公式计算毛利率将近80个点：
> "按照之前的公式计算都将近80个点了"

这显然超出了她的心理预期（70上下，75到天了）。她认为计算偏高，需要找到原因。

**基于公式流程分析的修正理解**：frog说"将近80个点"很可能指的是某个分板块的计算结果（如DRAM单独毛利推算74.8%），而非整体毛利率。整体57.08%相比2025年的40.21%，确实提升了16.87个百分点，涨幅巨大。

如果frog用"70上下"来指全年预期毛利率，57.08%是单季数据，全年可能回落，但frog说的"75到天了"显然是过于乐观的。

#### 第二层：分板块毛利率拆分

frog将业务拆分为四个板块，并分别估算毛利率：

**讨论中的板块占比假设：**

frog的假设占比（加总=100%）：
| 板块 | frog假设占比 |
|------|-------------|
| DRAM | 33% |
| NOR Flash | 40% |
| MCU | 20% |
| SLC NAND | 7% |

**与2025年报对比验证：**

| 产品 | 2025年报收入（亿元） | 年报占比 | frog假设占比 | 差异 |
|------|-------------------|---------|-------------|------|
| 存储芯片（合计） | 65.66 | **71.3%** | DRAM+NOR+NAND=80% | ❌ frog偏高 |
| MCU | 19.10 | **20.7%** | 20% | ✅ 基本吻合 |

**关键问题：年报不单独披露DRAM/NOR/NAND各自收入，只给存储芯片整体71.3%。**

frog的"DRAM 33% + NOR 40% + SLC NAND 7% = 80%"与年报存储芯片71.3%存在矛盾。可能的原因：
1. frog的占比是**存储芯片内部结构**而非整体占比（但表述为整体占比）
2. frog用的是**2026Q1预期结构**，DRAM占比已提升
3. frog的假设本身不自洽，需要修正

**合理推算存储芯片内部结构（基于行业数据和长鑫关联交易）：**

假设DRAM毛利率70%、NOR毛利率50%、SLC NAND毛利率50%，存储芯片整体毛利率42.84%：

```
存储芯片毛利 = DRAM占比×70% + NOR占比×50% + NAND占比×50% = 42.84%

若DRAM:NOR:NAND = 50:40:10（存储内部占比）：
  50%×70% + 40%×50% + 10%×50% = 35% + 20% + 5% = 60% ≠ 42.84%

若DRAM:NOR:NAND = 30:55:15（存储内部占比）：
  30%×70% + 55%×50% + 15%×50% = 21% + 27.5% + 7.5% = 56% ≠ 42.84%
```

存储芯片整体毛利率42.84%远低于DRAM 70%的假设，说明：
- **DRAM在存储芯片中占比可能低于30%**（2025年）
- 或者**DRAM毛利率远低于70%**（2025年可能只有50%左右）
- 2026年DRAM毛利率大幅提升是关键变化

**修正后的2025年存储芯片结构推算：**

| 板块 | 存储内部占比 | 整体占比 | 推算毛利 | 贡献 |
|------|------------|---------|---------|------|
| DRAM | 25%-30% | 18%-21% | 50%-55% | 9%-11.5% |
| NOR Flash | 55%-60% | 39%-43% | 45%-50% | 17.5%-21.5% |
| SLC NAND | 10%-15% | 7%-11% | 45%-50% | 3.5%-5.5% |
| **存储合计** | 100% | **71.3%** | - | **42.84%** |

**结论：frog的NOR 40%假设与年报基本吻合（年报存储芯片71.3%，NOR约占其中55%-60%即39%-43%），但DRAM 33%对2025年偏高，对2026Q1可能准确。**

**长鑫关联交易揭示的DRAM全年放量节奏：**

长鑫关联采购额 ≈ DRAM晶圆采购成本（长鑫是兆易DRAM独家代工厂）。

```
2025年基准：
  长鑫关联采购 = 11.82亿，DRAM整体占比约18%-21%
  存储芯片毛利率42.84%，推算DRAM毛利约50%-55%（远低于2026年的70%+）

Q1数据：
  DRAM占比 = 33%（公司报告），较2025年大幅提升
  Q1长鑫关联采购 = 5.81亿

全年数据：
  年度关联交易预计额度 = 57.11亿
  剩余季度平均 = (57.11 - 5.81) / 3 = 17.10亿/季度
```

**关键发现：Q1 DRAM占比33%是准确的，但全年DRAM占比将大幅提升。**

长鑫关联交易数据揭示：
- Q1仅完成年度预计额的10.2%（5.81/57.11）
- 后续三个季度DRAM将放量，平均每季度关联采购约17.1亿
- 这意味着全年DRAM收入占比可能从Q1的33%提升到50%+

**frog的33%假设对Q1是准确的，但用于全年推演会低估DRAM贡献。**

**讨论中的毛利率假设：**

| 板块 | 讨论假设毛利 | 2025年报实际毛利 | 备注 |
|------|-------------|----------------|------|
| DRAM | 70%（全年） | 含在存储42.84%中 | 年报不单独披露DRAM毛利 |
| 非DRAM综合 | 51.89%-53.22% | - | frog推算值，53.22%为简单加法，51.89%经CPA分母效应修正 |
| MCU | 35%（去年年报） | **35.82%** | 基本吻合！ |

#### 第三层：涨价传导与毛利分配

这是推演中最核心且最具争议的部分：

**frog的涨价分配假设：**
- NOR/MCU涨价约15个点
- 上游代工厂（中芯国际）分得2/3涨幅（约10个点）
- 下游兆易分得1/3涨幅（约5个点）

**由此推导非DRAM毛利率：**

简单加法：48.22% + 5pp = 53.22%

但frog最终采用了51.89%。根据公式流程分析的CPA精算修正，53.22%→51.89%的核心修正是**分母效应**——涨价后收入基数从P变为1.15P，正确做法是重新计算 `(新收入 - 新成本) / 新收入`，而非简单加5pp：

```
新毛利率 = (1.15P - 1.07C) / (1.15P) = 0.5960 / 1.15 = 51.83% ≈ 51.89%
```

frog也承认："公式有瑕疵"，因为量能不一样，这是粗略计算。详见《兆易创新Frog推演公式流程分析》第六章。

## 三、推演与财报实际数据对比

### 3.1 2026Q1整体毛利率验证

| 指标 | frog讨论预期 | 财报实际 | 验证结果 |
|------|-------------|---------|---------|
| 整体毛利率 | "将近80个点"/"70上下" | **57.08%** | frog预期偏高 |

**基于公式流程分析的修正理解**：frog说"将近80个点"很可能指的是DRAM单独毛利推算结果（第3步反推得74.8%），而非整体毛利率。整体57.08%相比2025年的40.21%，确实提升了16.87个百分点，涨幅巨大。

### 3.2 分板块毛利率验证

年报中**不单独披露**DRAM、NOR、NAND各自的毛利率，只给出存储芯片整体42.84%。

frog推算的核心逻辑是扣除法反推（详见公式流程分析第1-3步）：
1. 已知整体毛利率 = 57.08%（2026Q1）
2. 假设DRAM毛利率 = 70%（保守估计，机构数据74.8%）
3. 假设DRAM占比 = 33%
4. 反推非DRAM综合毛利

**用2026Q1实际数据反推（DRAM占比33%，公司报告）：**

```
整体毛利 = DRAM占比 × DRAM毛利 + 非DRAM占比 × 非DRAM毛利

57.08% = 33% × 70% + 67% × X

57.08% = 23.1% + 67% × X

67% × X = 57.08% - 23.1% = 33.98%

X = 33.98% / 67% = 50.72%
```

**frog推算的非DRAM毛利为51.89%-53.22%，与反推结果50.72%非常接近！** Q1阶段frog的DRAM占比33%假设是准确的（公司报告数据），反推逻辑成立。

**但全年推演需考虑DRAM占比提升：**

长鑫关联交易数据揭示Q1仅完成年度预计10.2%，后续季度DRAM放量将大幅提升占比：

```
假设Q2-Q4 DRAM关联采购平均17.1亿/季度，DRAM毛利率70%：
  Q2-Q4 DRAM收入 ≈ 17.1 / (1-0.7) × 0.75（封测系数）≈ 42.75亿/季度
  非DRAM收入假设保持Q1水平 ≈ 28.06亿/季度
  DRAM占季度收入比 ≈ 42.75 / (42.75 + 28.06) ≈ 60.4%

全年DRAM占比推算：
  DRAM收入 ≈ 13.82 + 42.75×3 ≈ 142.07亿
  非DRAM收入 ≈ 28.06×4 ≈ 112.24亿
  全年DRAM占比 ≈ 142.07 / (142.07 + 112.24) ≈ 55.9%
```

**全年DRAM占比可能从Q1的33%提升到55%+，这是全年毛利推演的关键变量。**

### 3.3 MCU毛利率验证

| 指标 | frog讨论 | 2025年报实际 | 验证 |
|------|---------|-------------|------|
| MCU毛利率 | "35上下" | **35.82%** | 基本吻合 |

frog准确引用了去年年报MCU毛利35%的数据。讨论中她发现机构测算与35%有矛盾：
> "这个毛利不对啊，去年的年报，mcu的毛利只有35上下"

这个质疑是合理的——如果MCU在2026Q1涨价15个点（其中分得5个点），毛利率应提升到约40-41%，而非维持35%。

### 3.4 DRAM毛利率验证

frog按70%保守估算DRAM全年毛利，机构数据74.8%。

2025年报描述DRAM毛利率"季度环比改善明显"，但没有单独披露。2026Q1存储芯片整体毛利率大幅提升（从42.84%到推算的更高水平），支持DRAM毛利确实在70%左右的推断。

### 3.5 53.22%→51.89%修正的财报验证

根据公式流程分析的CPA精算修正，frog从53.22%调到51.89%的核心修正是分母效应：

```
售价涨15%，代工成本涨7%时：
  新毛利率 = (1.15P - 1.07C) / (1.15P) = 51.83% ≈ 51.89%
```

用Q1实际数据反推非DRAM毛利率为50.72%，frog的51.89%比53.22%更接近实际值，说明CPA分母效应修正确实提高了推算精度。

## 四、2026年全年毛利推演路径

### 4.1 frog的全年推演框架

讨论中frog推算的是全年毛利，而非仅一季度。她的推演步骤：

1. **以一季度为基数**，用实际Q1数据打底
2. **叠加后续涨价**：NOR二季度涨更凶，MCU也涨价
3. **分配涨价传导**：1/3给兆易，2/3给上游代工
4. **CPA分母效应修正**：53.22%→51.89%（详见公式流程分析第六章）
5. **量增参数**：原预计8%，因Q1有新基数不再使用
6. **关联交易锚定年化**：57.11÷5.81=9.83倍，而非简单Q1×4

### 4.2 用财报数据修正全年推演

**已知条件：**
- 2026Q1毛利率 = 57.08%
- 2025全年毛利率 = 40.21%
- 存储芯片涨价持续（NOR二季度涨更凶、MCU涨价15个点）
- DRAM毛利率约70%
- **长鑫关联交易：Q1实际5.81亿，全年预计57.11亿**
- **DRAM占比：Q1为33%（公司报告），全年将大幅提升**

**全年毛利推演（长鑫关联交易修正版）：**

核心变化：DRAM业务在Q1仅完成年度10%，后续季度将大幅放量。全年DRAM占比从Q1的33%提升至55%+，高毛利DRAM占比提升将拉高全年综合毛利率。

| 板块 | 2025年报占比 | Q1占比 | 全年占比（修正） | Q1毛利 | 全年毛利假设 | 趋势说明 |
|------|------------|--------|----------------|--------|------------|---------|
| DRAM | 18%-21% | 33%（公司报告） | **55%-60%** | 70% | 68%-72% | 放量是最大变量 |
| NOR | 39%-43% | ≈25% | **18%-22%** | ≈50% | 55%-58% | 涨价传导，占比被动下降 |
| MCU | 20.7% | ≈20% | **11%-14%** | ≈40% | 42%-45% | 涨价+车规，占比被动下降 |
| SLC NAND | 7%-11% | ≈5% | **4%-6%** | ≈50% | 55% | 涨价受益，基数低 |
| 传感器+模拟 | 7.8% | ≈7% | **4%-6%** | ≈25% | 28%-32% | 增速慢于DRAM |

注：2025年报占比基于"存储芯片71.3%内推算DRAM/NOR/NAND比例"（见2.1节分析），MCU/传感器/模拟为年报直接数据。全年占比变化的核心驱动是DRAM放量导致其他板块占比被动稀释，但绝对收入仍在增长。

**全年综合毛利率推算（加权平均）：**

```
情景1（DRAM占比55%，DRAM毛利70%）：
  综合毛利 = 55%×70% + 20%×56% + 12%×43% + 5%×55% + 8%×28%
           = 38.5% + 11.2% + 5.16% + 2.75% + 2.24%
           = 59.85%

情景2（DRAM占比60%，DRAM毛利72%）：
  综合毛利 = 60%×72% + 18%×57% + 11%×44% + 5%×55% + 6%×30%
           = 43.2% + 10.26% + 4.84% + 2.75% + 1.8%
           = 62.85%

情景3（保守：DRAM占比50%，DRAM毛利68%）：
  综合毛利 = 50%×68% + 22%×55% + 14%×43% + 6%×55% + 8%×28%
           = 34% + 12.1% + 6.02% + 3.3% + 2.24%
           = 57.66%
```

**全年毛利率推演区间：57.7%-62.9%，中值约60%。**

**与frog推算对比：**
- frog按DRAM占比33%推算全年≈55%+，严重低估了DRAM放量对全年毛利的拉动
- 长鑫关联交易57.11亿vs Q1实际5.81亿，说明全年DRAM业务量是Q1的近10倍
- DRAM高毛利（70%+）占比从33%提升到55%-60%，是全年毛利高于Q1的核心驱动

### 4.3 关键不确定性

| 不确定性 | 影响方向 | 说明 |
|----------|---------|------|
| DRAM全年占比 | **↑↑** | 长鑫57.11亿年度额度vs Q1仅5.81亿，DRAM放量是最大变量 |
| DRAM放量节奏 | **↑↑** | Q2-Q4平均17.1亿/季度是否可实现？产能爬坡和需求匹配 |
| DRAM毛利率 | ↑ | 量大后是否维持70%+？DDR5 vs DDR4产品结构变化 |
| NOR涨价幅度 | ↑ | 二季度涨更凶，frog原图中推算可能低估 |
| MCU涨价分配 | ↑ | 蓝小娜说主流系列20%涨幅，frog按15%算偏保守 |
| 代工涨价传导 | ↓ | 中芯涨价会压缩兆易毛利空间 |
| 非DRAM占比被动压缩 | **↓** | DRAM占比提升→NOR/MCU占比被动下降，但绝对值仍增长 |
| 长鑫关联交易完成率 | **?** | 57.11亿是预计额度，实际完成可能低于100% |

## 五、核心结论

### 5.1 frog推演评价

| 维度 | 评价 | 说明 |
|------|------|------|
| **推演框架** | ✅ 优秀 | 分板块拆分+扣除法反推+涨价传导+CPA分母效应修正，逻辑清晰 |
| **MCU毛利引用** | ✅ 准确 | 35%与年报35.82%吻合 |
| **Q1 DRAM占比** | ✅ 准确 | 33%与公司报告一致 |
| **DRAM毛利假设** | ✅ 合理 | 70%保守估计，与推算接近 |
| **非DRAM毛利推算** | ✅ Q1阶段准确 | 51.89%-53.22%，与Q1反推50.72%接近 |
| **53.22%→51.89%修正** | ✅ 符合CPA规则 | 分母效应修正，售价涨15%+成本涨7%→51.83%≈51.89% |
| **全年DRAM占比** | ❌ 严重低估 | 用Q1的33%推全年，未考虑DRAM放量节奏 |
| **整体毛利预期** | ❌ 偏高 | "70-80"远超实际57.08%，口径可能有误 |
| **全年毛利推算** | ⚠️ 偏保守 | 55%+未考虑DRAM占比提升的拉动效应 |
| **涨价传导分配** | ⚠️ 需验证 | 1/3/2/3分配是假设，实际代工涨价比例未知 |

### 5.2 财报验证结论

1. **2026Q1毛利率57.08%**，较2025全年40.21%提升16.87个百分点，涨幅巨大但远低于frog"70-80"的预期
2. **frog的分板块反推法对Q1有效**，非DRAM毛利推算与实际反推结果误差仅1-2个百分点
3. **MCU毛利35%**的引用完全准确
4. **53.22%→51.89%的修正符合CPA分母效应规则**，售价涨15%、成本涨7%时，新毛利率=(1.15P-1.07C)/1.15P=51.83%≈51.89%，比简单加法53.22%更接近Q1反推值50.72%
5. **frog最大的局限是未考虑DRAM全年放量节奏**——Q1 DRAM占比33%是准确的，但长鑫关联交易57.11亿年度额度揭示全年DRAM占比将大幅提升
6. **全年毛利率预计在57.7%-62.9%区间（中值约60%）**，高于frog的55%+推算，核心驱动是DRAM高毛利业务占比从Q1的33%提升至55%-60%

### 5.3 frog推演中的核心偏差

1. **"将近80个点"**：基于公式流程分析，这很可能是DRAM单独毛利推算结果（第3步反推得74.8%），而非整体毛利率，实际Q1整体毛利率57.08%
2. **DRAM占比用Q1数据推全年**：Q1 DRAM占比33%是准确的，但长鑫关联交易数据表明全年占比将大幅提升至55%-60%，这是全年推演最大的遗漏
3. **量增8%严重低估**：Q1营收同比增119%，远超8%的量增预期，说明涨价+量增的双重驱动远超预判
4. **全年毛利55%+偏保守**：未考虑DRAM占比提升对综合毛利的拉动效应，修正后全年毛利中值约60%

### 5.4 长鑫关联交易的核心启示

长鑫关联交易是推演全年毛利率的**最关键锚点**，frog讨论中完全缺失此数据：

| 指标 | 数据 | 推演意义 |
|------|------|---------|
| Q1实际关联采购 | 5.81亿 | 对应Q1 DRAM占比33%，与公司报告一致 |
| 全年预计额度 | 57.11亿 | 对应全年DRAM占比55%-60%，是Q1的近10倍 |
| Q1完成率 | 10.2% | DRAM放量集中在Q2-Q4，平均17.1亿/季度 |
| 2025年实际 | 11.82亿 | 2026年预计是2025年的4.8倍 |

**长鑫关联交易57.11亿意味着兆易创新2026年DRAM业务将爆发式增长，这是全年毛利率超越Q1（57.08%）的核心驱动。**
