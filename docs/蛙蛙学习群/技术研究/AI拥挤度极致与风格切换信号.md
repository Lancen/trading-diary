## 一、拥挤度指标：5%个股成交占比

核心指标：**5%个股总成交额占整个市场的比例**。

统计2006年以来该指标走势，**45%是明显的风险边界**。

### 历史上5次突破45%的案例

| 时间 | 市场变化 | 类型 |
|------|----------|------|
| 2007年10月 | 6124大顶，牛熊转换 | 牛熊转换 |
| 2008年10月 | 1664大底，熊牛转换 | 牛熊转换 |
| 2015年1月 | 价值切回成长，6月见顶 | 风格切换 |
| 2018年1月 | 全面走熊 | 牛熊转换 |
| 2021年2月 | 茅指数见顶，风格切换到宁指数 | 风格切换 |

**5次突破，均对应市场大变化：3次牛熊转换，2次风格切换。**

当前该指标已超过45%风险线，预警信号再次出现。

## 四、与2021年2月茅指数瓦解的类比

2021年2月极致瓦解后的走法：

> 部分抱团股筑顶 → 主线缩圈 → 低位方向见底 → 风格逐步扩散

当时：
- 主线从**茅指数**缩圈到**宁组合**（增速更高的宁组合接过旗帜）
- 同时，**微盘股指数**和**红利指数**正式见底，后来持续走牛，成为真正主角

当前盘面类似，但仅为预演，并非终局。谁能真正跑出来，还需时间验证。

## 六、策略建议

### 核心观察方向

- **中概股** → 对应AI应用
- **储能** → 对应新能源
- **电力** → 对应红利
- **创新药** → 对应消费
- **微盘股** → 对应小盘风格

### 操作原则

- **等**：一等指数企稳，二等新主线出现
- 若真是风格转换，不会一蹴而就，市场需要时间形成共识
- **继续以防守为主，保持谨慎策略**
